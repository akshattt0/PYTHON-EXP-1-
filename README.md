## Theory

Python is a high-level, interpreted programming language known for its simplicity and readability.  
It can be executed using different platforms such as the Python shell, Jupyter Notebook, and various IDEs.

Python supports two main execution modes:
- **Interactive Mode**, where statements are executed line by line
- **Script Mode**, where a complete program is written and executed at once

Comments are used in Python to explain code and improve clarity. Single-line comments begin with the `#` symbol, while multi-line comments are written using triple quotes.

Variables are used to store data values and do not require explicit declaration of data types due to Pythonâ€™s dynamic typing feature. Python provides several built-in data types such as integers, floating-point numbers, strings, and boolean values.

Operators allow calculations and logical operations to be performed on data. Python supports arithmetic, relational, logical, assignment, and bitwise operators.

Input and output operations enable interaction between the user and the program. The `input()` function is used to accept input from the user, while the `print()` function is used to display output.
